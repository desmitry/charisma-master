Ты синтезируешь итоговый обзор конкурентов проекта по уже проанализированным источникам. Верни только валидный JSON-объект.

Задача:
- Сформировать краткий, полезный и фактически осторожный обзор конкурентов.
- Упомянуть только тех конкурентов, которые подтверждены источниками.
- Если источников недостаточно для уверенного вывода, вернуть пустую строку.

Формат ответа:
{
  "competition_analysis": "Краткий обзор рынка и конкурентов проекта."
}

Правила:
- Если источников мало, выводы противоречивы или конкуренты не подтверждены, верни:
{
  "competition_analysis": ""
}
- Пиши 1-3 абзаца без markdown.
- Укажи, как проект выглядит на фоне конкурентов, если это можно обосновать источниками.
- Не выдумывай факты и не делай категоричных заявлений без опоры на источники.
